## UI Liability & Informational Framing Update (Home.jsx)
### Summary
This update refines the Home page user interface language to ensure clear informational-only framing and reduced legal liability exposure.

### What Changed
- Reframed descriptive language to avoid implying legal advice or legal services
- Added a clear informational disclaimer on the Home page
- Clarified that CAPtenant provides educational and organizational tools only
- Explicitly scoped the Voice Assistant as a support and information tool
- Clarified that authoritative content is provided in English and French only
- Maintained neutral, non-directive wording throughout the UI

### What Did NOT Change
- No functional or behavioral changes
- No routing changes
- No backend or API changes
- No state, logic, or data flow modifications
- No user interaction changes

### Intent
These changes are strictly presentational and compliance-oriented, ensuring:
- Clear user expectations
- Reduced risk of misinterpretation as legal advice
- Alignment with informational-only positioning across the app

### Scope
- Affected file: `src/pages/Home.jsx`
- Change type: UI text and framing only
